# 🚶 Случайное блуждание (Random Walk)

Визуализация случайного блуждания с помощью Python и Matplotlib! 📊

## 🔍 О проекте

Этот проект демонстрирует концепцию **случайного блуждания** — математической модели, где объект перемещается в случайных направлениях. Программа генерирует путь из заданного количества точек, а затем визуализирует его в виде графика. 🗺️

## ✨ Возможности

- 🎲 Генерация случайного блуждания с заданным количеством точек (по умолчанию — 5 000).
- 🖌️ Визуализация пути с градиентной раскраской (от начала к концу).
- 🟢 Выделение начальной точки (зелёный цвет).
- 🔴 Выделение конечной точки (красный цвет).
- 🧹 Очистка осей графика для более эстетичного вида.
- 🔁 Возможность создавать несколько блужданий подряд.

## 🛠️ Используемые технологии

- **Python** 3.x 🐍
- **Matplotlib** — для визуализации данных 📈
- **random** — для генерации случайных значений 🎰

## 💾 Файлы проекта

- `random_walk.py` — класс `RandomWalk` для генерации случайного блуждания.
- `rw_visual.py` — скрипт для визуализации блуждания с использованием Matplotlib.

## 🔧 Установка и запуск

### Требования

Убедитесь, что у вас установлен Python 3.x и библиотека Matplotlib:

```bash
pip install matplotlib
```

### Запуск

1. Клонируйте репозиторий:

   ```bash
   git clone https://github.com/dshebeko20/random_walk.git
   ```

2. Перейдите в директорию проекта:

   ```bash
   cd random_walk
   ```

3. Запустите скрипт визуализации:

   ```bash
   python rw_visual.py
   ```

4. После отображения графика программа спросит:

   ```
   Make another walk? (y/n):
   ```

   Введите `y`, чтобы создать новое блуждание, или `n`, чтобы завершить программу.

## ⚙️ Как это работает

1. **Генерация шагов**:
   - Для каждой оси (`X` и `Y`) выбирается направление (`+1` или `-1`) и расстояние (`0–4`).
   - Нулевые перемещения (когда `x_step == 0` и `y_step == 0`) игнорируются, чтобы избежать «зависания» на месте.
2. **Построение пути**:
   - Каждая новая точка вычисляется как предыдущая точка + шаг.
   - Координаты сохраняются в списках `x_values` и `y_values`.
3. **Визуализация**:
   - Точки наносятся на график с цветовой картой `Blues`, где цвет меняется от начала (`синий`) до конца (`светло‑синий`).
   - Начальная точка (`0, 0`) выделяется зелёным цветом.
   - Конечная точка выделяется красным цветом.
   - Оси графика скрываются для лучшего восприятия пути.
